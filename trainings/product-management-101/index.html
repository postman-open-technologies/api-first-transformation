Maximizing Value With Partner & Public APIs

Confidently delivering the experience API consumers expect today using the Postman API platform.

It has become essential to doing business in the modern digital landscape that we are perpetually maximizing the value produced through our partner and public APIs. This narrative and supporting curriculum and workspace are designed to help you understand how you can maximize business value with your customers securely by publishing external APIs and treating them as a product.

This module intends to help set the stage for what you will learn throughout this training, with the following modules:

  • Portals Are the Status Quo In Many Enterprises - Starting from the current state of API engagement with consumers that is centered around the publicly available API portal.
  • Documentation For Our APIs Is Always Critical - Centering our engagement with consumers around essential API documentation while demonstrating how the space is changing.
  • Collections as a Modular Unit of Engagement - Helping us be more modular in how we make our APIs available through the usage of executable and sharable Postman Collections.
  • Workspace Show Our Homework in Real-Time - Revealing the importance of having a dedicated workspace around each of your APIs to properly engage with your consumers.
  • Networks Are the Future of API Engagement - Exposing your APIs and the workspace behind them using partner or public workspaces to help focus engagement with consumers.
  • Run In Postman Meets Developers Where They Are - Empowering you to use embeddable Run in Postman buttons to meet developers where they are as part of their work.
  • Investing in Developer Experience is What Matters - Making your API available to consumers is much more than just a portal and docs and requires a focus on the overall experience.
  • Measuring Meaningful Engagement with Consumers - Collections provide a unique opportunity to measure very modular and meaningful with your API consumers via workspaces.
  • Leveraging and Learning with Feedback Loops - Utilizing comments to focus your feedback loops with consumers, helping API producers learn from how consumers are using APIs.

This training will focus on demo workspace, APIs, collections, and other supporting developer resources to help make this narrative a hands-on experience. You should walk away from this training module with a better awareness of how engaging with API consumers has evolved in recent years, and have a base understanding of how you can use Postman to maximize value with your partner and public APIs.

Learnings:

  • How Postman maps to your existing portal investment.
  • Treating your external APIs as a product is important.
  • Investing in developer experience matters the most.
  • Collections allow for a composable approach to business.
